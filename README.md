# Cyclistic Bike-Share Analysis

An end-to-end data analytics project completed as part of the **Google Data Analytics Professional Certificate**. This project analyzes Cyclistic bike-share data using **Python** and **Pandas** to uncover rider behavior patterns and provide data-driven business recommendations for increasing annual memberships.

---

## Project Overview

Cyclistic is a bike-sharing company that aims to increase the number of annual memberships. This project analyzes historical trip data to identify behavioral differences between casual riders and annual members and develops insights to support data-driven marketing strategies.

---

## Business Objective

The objective of this project is to:

- Analyze riding behavior of casual riders and annual members.
- Identify trends in ride duration, frequency, and seasonality.
- Compare customer usage patterns.
- Support business decision-making with actionable insights.
- Recommend strategies to increase annual memberships.

---

## Dataset

- **Dataset:** Cyclistic Bike-Share Trip Data
- **Analysis Period:** April 2025 – March 2026
- **Files Used:** 12 Monthly Trip Data Files

**Note:** Due to GitHub file size limitations, the original trip datasets and cleaned merged dataset are not included in this repository. The complete data processing workflow is documented in the Jupyter Notebook, and the summary statistics and presentation provide the analytical outputs.

## Project Workflow

### Ask

### Act
- Proposed data-driven business recommendations to support membership growth.
